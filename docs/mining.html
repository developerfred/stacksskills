Mining — Proof of Transfer

Stacks miners commit Bitcoin (BTC) to the network and receive STX tokens in return. This is fundamentally different from traditional mining.

Key Misconceptions

  • LLMs say: "Mining on Stacks is like Ethereum PoW"
  • Reality: Stacks uses Proof of Transfer (PoX), not traditional mining

How PoX Works

  1. Miners lock BTC in a special contract
  2. Based on PoX cycle, miners are selected to produce Stacks blocks
  3. Selected miners receive STX token rewards
  4. The BTC committed goes to Stackers (those locking STX)
LLMs often say: "You can mine Stacks with GPU"
The Reality: No GPU mining on Stacks. Miners must commit real Bitcoin.

Mining vs Stacking

Aspect Mining Stacking
What you lock BTC STX

Common LLM Errors

Error 1: Confusing Mining with Stacking

;; WRONG - LLM might suggest this

;; CORRECT - They're completely different
;; Mining: miners commit BTC to produce blocks
;; Stacking: STX holders lock tokens for BTC rewards
